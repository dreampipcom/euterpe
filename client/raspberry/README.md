# Euterpe - DreamPip's Audio Broadcast Platform (Pocket Client)

What: Portable straming station, Raspberry PI.

## Getting Started

Install on RPI
```
chmod +x ./init.sh
./init.sh
```

Start Streaming
```
1. Plug audio source (Jack).
2. Connect to internet.
3. Power up Raspberry PI.
```
